# Pac-Man Web Game

A classic Pac-Man game built with HTML5, CSS, and JavaScript.

## 🎮 Play Now

Visit: **https://iceeer.github.io/pacman-web-game/**

## 🕹️ How to Play

- Use **Arrow Keys** to move Pac-Man
- Eat all the pellets to win
- Avoid the ghosts!
- Power pellets let you eat ghosts for bonus points

## 🎯 Features

- Classic Pac-Man gameplay
- Keyboard controls (Arrow keys)
- Score system
- 4 ghosts with AI
- Power pellets
- Win/lose conditions
- Lives system (3 lives)

## 🏗️ Tech Stack

- HTML5 Canvas
- CSS3
- Vanilla JavaScript

## 📁 Files

- `index.html` - Main HTML file
- `style.css` - Game styling
- `game.js` - Game logic

## 🎨 Game Controls

| Key | Action |
|-----|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |

## 📝 Scoring

- Regular pellet: 10 points
- Power pellet: 50 points
- Ghost (when scared): 200 points

Enjoy the game! 🎮
